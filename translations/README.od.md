# GitHub ସ୍ନାତକ - ୨୦୨୨

![୨୦୨୨-github-ସ୍ନାତକ-ସାମାଜିକ-କାର୍ଡ -1](/assets/GHG_Blog_1.jpg)

ଏହି ସଂଗ୍ରହାଳୟରେ **GitHub ସ୍ନାତକ ୨୦୨୨** ବର୍ଷ ପାଇଁ ପୁସ୍ତକ ଅଛି | ଏହି ସଂଗ୍ରହାଳୟକୁ ଏକ ଟାଣ ଅନୁରୋଧ ପ୍ରଦାନ କରି, ଆପଣ ୨୦୨୨ ର GitHub କ୍ଲାସରେ ଯୋଡିବାକୁ ଅନୁରୋଧ କରିପାରିବେ |

ମେ ୭ ସୁଦ୍ଧା ଟାଣ ଅନୁରୋଧ ସଫଳତାର ସହିତ ସଂଗ୍ରହାଳୟରେ ମିଶ୍ରିତ ହେଲେ, ପ୍ରଥମ ୭,୫୦୦ ଟାଣ ଅନୁରୋଧ କରିଥିବା ଛାତ୍ରଙ୍କୁ ମେଲରେ କଷ୍ଟମ୍ ଟ୍ରେଡିଂ କାର୍ଡ, ଷ୍ଟିକର୍ ଏବଂ ଚିଠି ଦିଆଯିବ |

## ଗୋପନୀୟତା ବିଜ୍ଞପ୍ତି 👀

ଧ୍ୟାନ ଦିଅନ୍ତୁ ଯେ ଆପଣ ଏହି ସଂଗ୍ରହାଳୟରେ ଯୋଗ କରୁଥିବା ସମସ୍ତ ସୂଚନା ସର୍ବସାଧାରଣରେ ଉପଲବ୍ଧ ହେବ |

- ଯଦି ତୁମେ ତୁମର ପୂର୍ଣ୍ଣ ନାମ ପ୍ରଦର୍ଶନ କରିବାରେ ଆରାମଦାୟକ ଅନୁଭବ କରୁନାହଁ, ତୁମେ ଏହା ବଦଳରେ ଏକ ଛୋଟ ନାମ କିମ୍ବା ଡାକନାମ ଅନ୍ତର୍ଭୂକ୍ତ କରିପାରିବ |

# କିଏ ଆବେଦନ କରିପାରିବେ 📝

ଆମେ ଯେକୌଣସି ଛାତ୍ରଙ୍କୁ ସ୍ନାତକ, କିମ୍ବା ସ୍ନାତକ କରିବାକୁ ଯୋଜନା କରୁଥିବା ୨୦୨୨ ମସିହାରେ ବର୍ଷ ପୁସ୍ତକରେ ଆବେଦନ କରିବାକୁ ଆମନ୍ତ୍ରଣ କରୁଛୁ | ଏଥିରେ ବୁଟକ୍ୟାମ୍ପ, କୋଡ୍ କ୍ୟାମ୍ପସ୍, ହାଇସ୍କୁଲ୍ ସ୍ନାତକ, ମାଷ୍ଟର ସ୍ନାତକ, ପିଏଛ.ଡି. ସ୍ନାତକ ଇତ୍ୟାଦି ଅନ୍ତର୍ଭୁକ୍ତ |

ଯୋଗ୍ୟତା ମାନଦଣ୍ଡ ହେଉଛି -

1. ଆପଣ GitHub ଛାତ୍ର ବିକାଶକାରୀ ପ୍ୟାକ୍ ସହିତ ଛାତ୍ର ଭାବରେ ଯାଞ୍ଚ ହୋଇଥିବେ  | ଏପର୍ଯ୍ୟନ୍ତ ପ୍ୟାକ୍ ର ଏକ ଅଂଶ ନୁହନ୍ତି କି? [ଏଠାରେ ଆବେଦନ କରନ୍ତୁ](https://education.github.com/discount_requests/student_application?utm_source=2022-06-11-GitHubGraduation).
2. ଆପଣ ଅତୀତର GitHub ସ୍ନାତକ କାର୍ଯ୍ୟକ୍ରମରେ ଅଂଶଗ୍ରହଣ କରିନାହାଁନ୍ତି |
3. ଆପଣ ୨୦୨୨ ମସିହାରେ ସ୍ନାତକ ଭାବରେ ପରିଚିତ |

GitHub ସ୍ନାତକରେ ଅଂଶଗ୍ରହଣ କରିବାର ଅଧିକ ଉପାୟ ଏବଂ ଆମର ସାମାଜିକ ଆକାଉଣ୍ଟରେ ବୈଶିଷ୍ଟ୍ୟ ହେବାର ସମ୍ଭାବନା ଖୋଜୁଛ କି?

ଆମେ ତୁମର ଶିକ୍ଷାବର୍ଷରେ ହାସଲ କରିଥିବା ଆଶ୍ଚର୍ଯ୍ୟଜନକ ଜିନିଷଗୁଡ଼ିକ ବିଷୟରେ ଶୁଣିବାକୁ ଚାହୁଁ ଏବଂ GitHub ତୁମର ଲକ୍ଷ୍ୟ ପୂରଣ କରିବାରେ କିପରି ସାହାଯ୍ୟ କଲା | କିଛି ସମୟ ନିଅ ଏବଂ ଏକ ଭିଡିଓ ରେକର୍ଡ କରି କିମ୍ବା ଏକ ବାର୍ତ୍ତା ଲେଖି ଆମ, ତୁମର ଶିକ୍ଷକ, ଏବଂ ସହପାଠୀମାନଙ୍କ ସହିତ ତୁମର କାହାଣୀ ଅଂଶୀଦାର କର |

[କିପରି ଅଂଶଗ୍ରହଣ କରିବେ](https://drive.google.com/file/d/1AcgUKLXx6WIC5s4eanzOfj8EsiYHARrt/view?usp=sharing)

ତୁମେ କ’ଣ କହିବାକୁ ଚାଁହୁଛ, ଆମେ ଶୁଣିବାକୁ ଆଗ୍ରହୀ ଅଟୁ, ଏବଂ ତୁମେ ଆମ ସମ୍ପ୍ରଦାୟର ଏକ ଅଂଶ ହେଇଥିବାରୁ ଆମେ କୃତଜ୍ଞ 💖

ପ୍ରଥମ ୭,୫୦୦ ସଫଳତାର ସହିତ ମିଶ୍ରିତ PR ଗୁଡିକ, ମେଲରେ ସେମାନଙ୍କର GitHub ସ୍ଥିତି ସହିତ ଏକ କଷ୍ଟମ୍ ହୋଲୋଗ୍ରାଫିକ୍ ଡେଭେଲୋପେର ଟ୍ରେଡିଂ କାର୍ଡ ପ୍ରଦାନ କରାଯିବ |

ଏହାର ଅର୍ଥ କ’ଣ? ଏକ ଟ୍ରେଡିଂ କାର୍ଡ ସୃଷ୍ଟି କରିବାକୁ ଆମେ ଆପଣଙ୍କର ସାର୍ବଜନୀନ GitHub ପ୍ରୋଫାଇଲ୍ ସୂଚନା ବ୍ୟବହାର କରିବୁ | ଆପଣଙ୍କର ଟ୍ରେଡିଂ କାର୍ଡ ଆପଣଙ୍କୁ ସର୍ବୋତ୍ତମ ପ୍ରତିଫଳିତ କରିବାକୁ ନିଶ୍ଚିତ କରିବାକୁ, ଦୟାକରି ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ଆପଣଙ୍କର GitHub ପ୍ରୋଫାଇଲ୍ ଚିତ୍ର ଏବଂ ବାୟୋ ଅପ୍ ଟୁ ଡେଟ୍ ଅଟେ ଏବଂ ଆପଣ କାର୍ଡରେ ଯାହା ଦେଖାଇବାକୁ ଚାହାଁନ୍ତି |
